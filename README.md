# Reader Vault Pro

Mobile + web app for saving and reading long-form text offline. Built with Capacitor (iOS/Android) and Vite.

## What it does

- **Load** text from a file (PDF/TXT), paste, or URL — URLs are fetched and stripped of HTML automatically.
- **Trim** the loaded text before saving, with a built-in find tool.
- **Save** named guides locally on the device with progress tracking.
- **Read** with a fullscreen reader, theme switching (dark/light/contrast), and per-word color highlighting.
- **Export/Import** your entire library as an encrypted link via PrivateBin. Works across devices. Links expire in 1 week.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Mobile (Capacitor)

```bash
npx cap sync
npx cap open ios      # or android
```
